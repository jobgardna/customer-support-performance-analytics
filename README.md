# 📊 Customer Support Performance Analytics Dashboard

## 🚀 Project Overview

This project presents an end-to-end Customer Support Performance Analytics solution built using SQL, Power BI, Excel, and Python.

The objective was to analyze operational performance metrics in a customer care environment to identify drivers of repeat contacts, improve resolution efficiency, and enhance customer satisfaction.

---

## 🎯 Business Problem

Customer support centers often struggle with:

- High repeat contact rates
- Low First Contact Resolution (FCR)
- Increasing Average Handle Time (AHT)
- Inconsistent Customer Satisfaction (CSAT)
- Uneven agent productivity

Leadership teams need visibility into operational inefficiencies and actionable insights to improve service performance.

---

## 🗂 Dataset Structure

The dataset simulates a real-world support center and contains:

### Tickets Table
- ticket_id
- customer_id
- agent_id
- channel (Call, Chat, Email)
- issue_category
- created_date
- resolved_time
- repeat_flag

### Agents Table
- agent_id
- agent_name
- team
- hire_date
- shift

### Customer Feedback Table
- ticket_id
- csat_score
- nps_score

---

## 📌 KPIs Tracked

---

## 🔍 Key Insights

- Billing-related issues drive highest repeat rate.
- Higher AHT negatively correlates with CSAT.
- Chat channel shows stronger FCR performance.
- Newer agents have higher repeat rates.
